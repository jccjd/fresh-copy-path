# Copy Path Plugin for Fresh

A Fresh plugin that copies the current file path to the clipboard.

## Installation

Install via Fresh's package manager:
```
:pkg install https://github.com/YOUR_USERNAME/fresh-copy-path
```

Or clone manually:
```bash
cd ~/.config/fresh/plugins/packages
git clone https://github.com/YOUR_USERNAME/fresh-copy-path
```

## Usage

1. Open the command palette with `Ctrl+P`
2. Type `copy_path` and press Enter

The current file path will be copied to your clipboard, and a status message will show the copied path.

## Commands

- `copy_path` - Copy current file path to clipboard
